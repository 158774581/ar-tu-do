Getting Started

These instructions will get you a copy of the project up and running

Move to ROS Workspace

1 cd ros_ws

Build ROS packages

1 catkin_make

Run tests

1 catkin_make run_tests

Run nodes

1 source devel/setup.zsh
2 roslaunch dev.launch

Python setup

This project uses autopep8 for automatic code formatting. Before you first use it, you need to install it like this: 

sudo apt-get install pip
pip install --upgrade autopep8

Contributing

These instructions will help you contribute code to the project.

Git Hooks

To install the repos git hooks run

1 scripts/hooks/_install.sh

Format C++ and Python code

1 ./scripts/format/format-src.sh

The python code is formatted with autopep8. It is currently configured with the aggressive flags. Thus, you need to review the changes it made.
